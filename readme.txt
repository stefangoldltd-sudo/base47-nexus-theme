Clean, responsive WordPress theme with customizable layouts, navigation options, and Canvas Mode for custom HTML layouts.

== Description ==

Nexus is a clean, responsive WordPress theme designed for flexibility and compatibility. It provides customizable layout options while maintaining full WordPress functionality.

= Key Features =

* **Canvas Mode**: Optional full-width page template without theme styling interference
* **Customizer Integration**: Standard WordPress Customizer options for colors, fonts, and layouts  
* **Custom Header Options**: Logo and navigation configuration using standard WordPress settings
* **Multi-level Navigation**: Support for dropdown menus with WordPress menu system
* **WooCommerce Compatible**: Basic styling support for WooCommerce pages
* **Responsive Design**: Mobile-first approach with flexible layouts
* **Custom Page Templates**: Multiple page layout options including full-width
* **Widget Areas**: Standard sidebar and footer widget support

= Canvas Mode =

Canvas Mode is an optional page template that removes theme styling to allow custom HTML content to display without interference. This is useful for:
- Landing pages with custom styling
- HTML templates that need precise control
- Full-width layouts without theme constraints

Canvas Mode can be selected per-page using the page template dropdown in the WordPress editor.

= Customization Options =

The theme includes standard WordPress Customizer sections for:
- Site identity (logo, title, tagline)
- Colors and typography  
- Header and navigation layout
- Footer configuration
- Widget area management

= Navigation Features =

- WordPress menu system integration
- Multi-level dropdown support
- Mobile-responsive navigation
- Customizable menu locations

= WooCommerce Support =

Basic compatibility with WooCommerce including:
- Product page layouts
- Shopping cart integration  
- Checkout page styling
- Mobile commerce optimization

== Installation ==

1. Upload the theme files to `/wp-content/themes/nexus/`
2. Activate the theme through 'Appearance > Themes' in WordPress admin
3. Customize the theme via 'Appearance > Customize'
4. Configure menus in 'Appearance > Menus'
5. Set up widgets in 'Appearance > Widgets'

== Frequently Asked Questions ==

= What is Canvas Mode? =

Canvas Mode is a page template that removes theme styling to allow custom HTML content full control over the page layout. Select it from the page template dropdown when editing a page.

= Does it work with page builders? =

Yes, Nexus works with popular page builders like Elementor, Gutenberg, and others. Canvas Mode can be used when you need complete layout control.

= Is it mobile responsive? =

Yes, Nexus is built with responsive design principles and works well on all device sizes.

= Does it support WooCommerce? =

Yes, the theme includes basic WooCommerce compatibility for product pages, cart, and checkout.

= Can I customize the header? =

Yes, use the WordPress Customizer to configure your logo, site title, and navigation menus. Multiple header layout options are available.

= How do I set up navigation menus? =

Go to Appearance > Menus in your WordPress admin to create and assign menus to the theme's menu locations.
